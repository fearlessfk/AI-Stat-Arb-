AI Stat-Arb Research Platform

让长尾加密配对交易，从论文线索到盲测复盘形成闭环

这是一个基于多 Agent 协作的量化策略猎头与回测评估系统，聚焦 Top 50-500 中小市值代币的配对交易。 平台自动完成前沿研究检索、可行性盲审、策略代码转化与样本外归因复盘，尽量把人力留给真正值得投入的 Alpha 假设。

核心痛点

平台不是单纯找策略，而是专门解决统计套利投研里最昂贵、最容易过拟合的三段工作。

策略情报处理过载

持续扫描 arXiv、SSRN、GitHub 等来源，由模型抽取数学结构、依赖因子与前置假设。

长尾协整维度爆炸

在中小市值代币池中滚动筛选协整对，并给出价差驱动与 Narrative Driver 的可解释线索。

样本内漂亮、样本外塌方

用自动化盲审与 Walk-forward 验证降低未来函数、生存者偏差和情绪化过拟合。

四阶段 Agent Pipeline

前台当前重点展示的是策略准入和解释层，但整体设计已经按四阶段闭环组织。

Agent 1: Discovery Scraper

检索最新统计套利研究，提取公式、特征、假设和适用场景。

Agent 2: Feasibility Grader

结合手续费、延迟、数据可得性做逻辑盲审，提前拦截无法落地的方案。

Agent 3: Code Translator

把策略研报翻译成符合 BaseStrategy 约束的向量化 Python 代码。

Agent 4: Post-mortem Analyst

读取万行回测日志，识别失效归因并沉淀成可追踪的失败知识库。

策略输入

输入一段关于长尾协整套利的假设，平台会先拆解研究要素，再生成候选样本、证据和准入建议。

研究情报源

这一层对应 Agent 1 的输入来源。原型当前可接外部搜索，也能回退到本地研究样本模式。

学术研究

arXiv, SSRN, NBER, working paper, 统计套利与微观结构论文

开源社区

GitHub, Medium, Substack, Reddit, X, 开发者论坛

交易执行语境

交易所 API、Maker/Taker 费率、资金费率、订单簿深度、流动性约束

策略拆解结果

把自然语言转换成结构化研究对象，供后续检索、逻辑盲审与代码转化使用。

候选研究样本

展示与输入策略最接近的协整套利框架，并按综合准入评分排序。

进入下一轮建议

这里给出的不是收益承诺，而是“是否值得占用回测与工程资源”的研究准入建议。

检索证据

展示本次研究所参考的外部样本与可信度，便于人工复核和申请材料展示。

最近研究归档

每次分析都会留下可追溯记录，为后续失败归因报告和 Alpha 知识库打底。

评分逻辑

平台优先回答“值不值得做”，因此评分重点放在落地、风险与研究价值，而非样本内收益曲线。

数学严谨性 协整假设、价差构造和过滤条件是否自洽
执行可达性 手续费、延迟、滑点和交易深度是否允许真实执行
数据可获得性 是否能稳定获取 K 线、orderbook、资金费率与事件数据
失效风险 是否容易受 Rug Pull、叙事切换、流动性蒸发影响
研究杠杆 是否值得进入代码转化与 Walk-forward 盲测阶段
